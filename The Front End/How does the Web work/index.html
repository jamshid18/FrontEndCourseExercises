Brief introduction to the internet

The Internet is a vast network which enable us to connect with each other.

How does the internet work

Internet connect us and with the help of this connection we can talk, communicate, and interact

Internet Protocols

Internet Protocols are set of rules which are made in order to make connection

Ip Address

Ip Address is actually numbers that we use it so that we can see the location of the sender and receiver.

DNS

DNS stands for Domin Name System. It covert the host name into an Ip address

Packets(TCP/UDP)

TCP stands for Transmission Control Protocol. TCP make sure that the receiver get the massage. if there is any problem with the massage, it can ask from the sender to give him the information that it needs. UDP stands for User Datagram Protocol. UDP is smaller than TCP and we use it normally for online streaming.

HTTP

HTTP stands for Hyper Text Transfer Protocol. It is a set of rules which enable us to communicate with World Wide Web.

HTML

HTML stands for Hyper Text Mark Up Languages. It is a code that we use it in order to write the structure of a web page.

Routers

Routers are devices that we use to make connection between computers. Routers enable us to make easier connection and we will have less traffic on it.
